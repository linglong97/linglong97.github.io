Microfluidic Device

In this project, I was tasked to optimise the control of a soft particle in a Microfluidic system. Using a DAQ card, a computer, 4 pressure controllers interfaced with a 3D printed device I printed, I wrote and optimised MATLAB code to reduce the runtime of the system from 200ms to 30ms. Code is available here.

Project: University of Toronto
